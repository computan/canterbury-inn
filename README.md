# Catapult WordPress Framework

Catapult is Computan's own website framework, featuring

1. the Catapult Design System and 
2. the Catapult Development Platform (in WordPress). 

We use the Catapult framework to guide our strategy, design, content production, and development process.

This documentation is a guide to the Catapult WordPress Framework.
